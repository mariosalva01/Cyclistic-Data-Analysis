# Cyclistic-Data-Analysis
"Análisis de Big Data con Python (Pandas) para el caso práctico de Google Data Analytics sobre el sistema de bicicletas Cyclistic."
Estrategia de Crecimiento para Cyclistic 🚲
Este proyecto forma parte del Certificado Profesional de Análisis de Datos de Google. El objetivo es analizar cómo difieren los patrones de uso entre los ciclistas ocasionales y los miembros anuales para diseñar una estrategia de marketing que aumente las suscripciones anuales.
🛠️ Herramientas y Metodología

    Lenguaje: Python 3.
    Librerías: Pandas, Matplotlib y Seaborn.
    Entorno: Google Colab.
    Dataset: +5.5 millones de registros de viajes (Mayo 2025 - Abril 2026).
    Fases del Proyecto: Preguntar, Preparar, Procesar, Analizar, Compartir y Actuar.

📋 Resumen del Proceso (Paso 3 y 4)
Se procesaron 12 archivos CSV unificándolos en un único DataFrame global. Se realizó una limpieza profunda eliminando datos atípicos (viajes <1 min y >24h) y se crearon nuevas variables como ride_length y day_of_week.
💡 Hallazgos Principales

    Diferencia de duración: Los usuarios ocasionales realizan viajes un 60% más largos (media de ~20 min) frente a los socios (~12 min).
    Patrón de uso: Los socios dominan los días laborales (transporte), mientras que los ocasionales se concentran en fines de semana (ocio).
